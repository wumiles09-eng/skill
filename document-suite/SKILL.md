## Best Practices

### General Documents
1. **Clear Structure**: Use headings hierarchically
2. **Consistent Formatting**: Apply styles uniformly
3. **Professional Fonts**: Use standard business fonts
4. **White Space**: Don't crowd content
5. **Proofread**: Check for errors

### Presentations
1. **One Idea Per Slide**: Keep it focused
2. **Visual Over Text**: Use images and diagrams
3. **Large Text**: Minimum 24pt for body
4. **Consistent Theme**: Use templates
5. **Practice**: Rehearse timing

### Spreadsheets
1. **Organized Data**: Use table format
2. **Clear Headers**: Label all columns
3. **Formulas**: Document complex formulas
4. **Validation**: Add input validation
5. **Backup**: Save versions regularly

### PDFs
1. **Optimize Size**: Compress images
2. **Bookmarks**: Add navigation
3. **Metadata**: Include document info
4. **Secure**: Add protection if needed
5. **Test**: Verify on multiple devices
