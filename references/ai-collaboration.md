# AI + Human Collaboration Guide

## When to Use AI in Short Drama Creation

AI excels at:
- **Structure generation**: Episode outlines, beat sheets, pacing maps
- **Format conversion**: Novel → script, Chinese → American adaptation
- **Volume production**: First drafts of episodes from detailed outlines
- **Localization**: Cultural element translation, dialogue Americanization
- **Quality checking**: Consistency validation, format verification

AI struggles with:
- **Emotional nuance**: Subtext, grief, love — requires human calibration
- **Cultural sensitivity**: What's offensive changes rapidly; AI can miss context
- **Original voice**: Distinctive authorial style requires human input
- **Trend timing**: What's viral this week won't be in AI training data

## The AI-Assisted Workflow

### Phase 1: Concept (Human-Led)

Human provides:
- Core emotional concept
- Target audience specification
- Genre and tone preferences
- Any source material (novel, script, idea)

AI assists with:
- Genre template matching
- Audience insight validation
- Competitive analysis (what's performing now)

### Phase 2: Architecture (AI-Assisted)

Human provides:
- Character concepts and arcs
- Key emotional beats
- The "karma moment" vision

AI generates:
- 60-80 episode master beat sheet
- Character profile documents
- Pacing map with intensity scoring
- Card boundary cliffhanger designs

Human reviews and adjusts:
- Emotional rhythm validation
- Cultural sensitivity check
- Personal creative vision alignment

### Phase 3: Episode Writing (Collaborative)

For each batch of 5-10 episodes:

1. AI generates first draft from detailed beat sheet
2. Human reviews for:
   - Emotional authenticity
   - Cultural accuracy
   - Dialogue naturalness
   - Hook strength
   - Cliffhanger impact
3. Human rewrites key scenes (especially emotional beats)
4. AI checks format consistency
5. Final human polish

### Phase 4: Quality Validation (AI-Assisted)

AI runs automated checks:
- Format compliance (sluglines, dialogue indentation)
- Hook/cliffhanger presence per episode
- Consistency (character names, timelines, locations)
- Pacing analysis (conflict density per episode)

Human reviews:
- Emotional journey coherence
- Cultural sensitivity final check
- Voice consistency
- Binge loop effectiveness ("Do I need to see the next episode?")

## AI Prompt Best Practices for Short Drama

### For Episode Outlines

```
Write a 5-episode beat sheet for a contract marriage short drama.
Parameters:
- Target: American women 25-45
- Genre: Billionaire romance with revenge elements
- Protagonist: 28, interior designer, hiding debt
- Love interest: 35, tech CEO, needs wife for inheritance clause
- Antagonist: His ex-fiancée, socialite, wants him back
- Card boundary: Episode 10 (first payment point)
- Tone: Sharp dialogue, high tension, slow-burn romance
- Setting: Manhattan / Hamptons

For each episode include: Location, Characters, Central Conflict, 
Twist, Cliffhanger. Ensure episode 5 ends with a MAJOR cliffhanger.
```

### For Episode Drafts

```
Write Episode 7 of "The Contract Wife" short drama.
Context from previous episodes: [2-3 sentence recap]
This episode's beats: [from beat sheet]
Format: 中文短剧普通文本正文 (`场号 + 人物 + △动作 + 角色：台词 + 旁白/OS/VO`), 90-150 seconds when produced; no tables, quote boxes, code blocks, or separate dialogue/narration boxes
Requirements:
- Open with visual hook (no establishing shots)
- 2-3 scenes maximum
- End with unresolved cliffhanger
- Dialogue: sharp, American idiom, no translated-sounding lines
- Characters: SARAH (28, determined, hiding fear), JACK (35, cold 
  exterior, occasional warmth), VICTORIA (30, antagonist, polished 
  cruelty)
Setting: Manhattan penthouse
```

### For Cultural Adaptation

```
Adapt the following Chinese short drama scene for American audiences.
Original scene: [paste scene]
Adaptation requirements:
- Replace all Chinese cultural references with American equivalents
- Remove hierarchical language (bows, formal address)
- Ensure female lead has active agency
- Add clear consent markers for romantic elements
- Americanize setting to Manhattan
- Check for cultural red flags
Output: Adapted scene + list of changes made + rationale
```

## Common AI Pitfalls in Script Generation

| Pitfall | Example | Fix |
|---------|---------|-----|
| Translated-sounding dialogue | "You dare to refuse me?" | Rewrite: "You think you can say no?" |
| Over-explaining | Characters state their feelings directly | Show through action; use subtext |
| Cultural blind spots | Including concepts like "filial piety" | Replace with American equivalent |
| Passive female leads | Waiting for rescue | Give her actions, choices, plans |
| Generic descriptions | "She is beautiful" | "She enters like she owns the room — because she does" |
| Flat pacing | Same intensity every episode | Vary conflict density; build to peaks |
| Predictable structure | Problem → solution in same episode | Problem → complication → cliffhanger |

## The 70/30 Rule

In AI-assisted short drama creation:
- **70%** of structural work can be AI-generated (outlines, format, consistency)
- **30%** must be human-created (emotional beats, cultural nuance, voice)

The 30% human input determines whether the script is competent or exceptional.

## Evaluating AI-Generated Episodes

Score each episode on these dimensions:

| Dimension | AI Should Handle | Human Must Review |
|-----------|---------------|-------------------|
| Format compliance | Yes | Spot check |
| Hook presence | Yes | Quality check |
| Cliffhanger strength | Yes | Impact validation |
| Pacing structure | Yes | Rhythm feel |
| Dialogue naturalness | Partial | Full review |
| Emotional authenticity | No | Full review |
| Cultural accuracy | Partial | Full review |
| Character voice consistency | Yes | Full review |
| Binge loop effectiveness | Yes | Gut check |

## The Iteration Cycle

AI-assisted short drama benefits from rapid iteration:

1. Generate draft → 2. Human review → 3. Specific feedback → 4. AI revision → 5. Human polish

Each cycle should take minutes, not hours. The speed advantage of AI is lost if human review takes days.

**Time allocation per episode**:
- AI generation: 2-5 minutes
- Human review: 10-15 minutes
- AI revision: 2-3 minutes
- Human polish: 10-15 minutes
- **Total per episode: 25-40 minutes**

At this rate, a 70-episode series can be drafted, reviewed, and polished in 30-45 hours of active work.
