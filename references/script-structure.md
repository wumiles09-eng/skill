# 剧本标准格式：前述 + 正文

## 概述

每部剧本必须包含**前述**和**正文**两大部分。
- **前述**：剧本的"说明书"，供制作方、平台、投资人快速理解项目
- **正文**：按集展开的完整剧本，供拍摄制作使用

**参考来源**：国内50+部短剧剧本格式分析，涵盖漫剧、仿真人剧、AI真人剧等多种类型。

---

## 第一部分：前述（前置内容）

### 1. 剧名

```
《英文剧名》
《中文译名》
```

- 英文剧名：面向美国市场的官方名称
- 中文译名：供中国制作团队内部使用
- 示例：《THE CONTRACT WIFE》《契约妻子》

### 2. 简介（故事梗概）

**字数**：200-500字（中文）
**功能**：用3-5句话讲清楚"这是一个什么故事"
**要求**：
- 包含主要人物设定
- 核心冲突/矛盾
- 故事大致走向
- 结局方向

**写法公式**：
```
[主角] 是一个 [身份/状态] 的 [年龄] 岁 [女性/男性]，
因为 [激励事件]，她/他陷入了 [困境/目标]。
在 [对手/障碍] 的阻挠下，她/他通过 [行动]，
最终 [结果/成长]。
```

**示例**：
```
Sarah（32岁）是一位被丈夫背叛的前企业高管，离婚后一无所有。
她隐姓埋名进入前夫的新公司做实习生，凭借过人能力迅速崛起。
当她的身份即将曝光时，她必须在复仇与新生之间做出选择——
而那个一直暗中帮助她的神秘CEO，似乎知道她所有的秘密。
```

### 3. 看点（创意亮点 / 卖点）

**功能**：列出这部剧的"为什么值得看"，供平台和投流使用
**格式**：3-5个看点，每条一句话

**看点类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| **人设看点** | 主角的独特设定 | "白切黑女主：表面温柔实习生，实则是被抛弃的前妻" |
| **关系看点** | 核心关系的张力 | "契约婚姻变真爱：从互相利用到彼此救赎" |
| **情绪看点** | 观众的情绪体验 | "每3集一次身份危机，全程窒息式追更" |
| **反转看点** | 剧情反转设计 | "第30集真相揭露：帮助她的CEO竟是她的亲生父亲" |
| **题材看点** | 题材的稀缺性 | "首部AI仿真人剧级别的狼人Alpha题材" |
| **爽点** | 打脸/复仇设计 | "第50集当众揭穿：她用他的钱买了他的公司" |

**示例**：
```
看点：
1. 人设反差：温柔实习生 × 腹黑前妻，双面人生步步惊心
2. 契约变真爱：从利益交换到真心交付，甜虐交织
3. 层层反转：每10集一个真相揭露，60集共6次颠覆认知
4. 极致爽感：第50集女主当众收购前夫公司，打脸全场
5. Netflix级质感：冷调权谋美学 × 暖调爱情光，视觉极致
```

### 4. 人物小传

**功能**：让每个角色立起来，有记忆点，不脸谱化
**格式**：每个角色包含以下维度

#### 人物小传五维度

| 维度 | 内容 | 示例 |
|------|------|------|
| **外在** | 姓名、年龄、外貌特征、身份、人际关系 | Sarah, 32岁，银灰色西装外套，凌厉短发，前VP现为实习生 |
| **内在** | 性格、偏好、价值观、信仰 | 外冷内热，信奉"以牙还牙"，内心深处渴望被真正看见 |
| **状态** | 当前人生阶段、心理状态 | 离婚后的自我重建期，表面坚强，深夜崩溃 |
| **行动** | 核心驱动力、典型行为模式 | 用专业能力证明自己，遇到不公绝不沉默 |
| **变化** | 角色弧线：从什么到什么 | 从"复仇是唯一目标"到"我值得为自己而活" |

**小传写法要求**：
- 每个角色150-300字
- 突出"反差感"和"记忆点"
- 避免扁平化，每个角色都有复杂性
- 标注角色关系（谁和谁是什么关系）

**主角小传示例**：
```
【女主】SARAH / 萨拉（32岁）

外在：前跨国企业VP，离婚后隐姓埋名成为实习生。
银灰短发，永远穿大一号的西装（藏起曾经的光芒），
手腕上有一道旧伤疤（从不解释来历）。

内在：信奉"公平是争来的不是等来的"。
表面温顺听话，实则每一步都在计算。
对弱者有本能的保护欲（源于曾被抛弃的经历）。

状态：处于人生最低谷后的反弹期。
白天是完美实习生，晚上在出租屋里研究前夫的财报。
她告诉自己这是复仇，其实也在寻找"我还值得被爱"的证明。

行动：用三个月从实习生升到项目经理，
每一步都踩在公司的规则边缘——但不违规。
当发现有人被欺负时，她会出手，哪怕暴露自己。

变化弧线：从"我要让他后悔"（复仇驱动）
到"我要让自己骄傲"（自我驱动），
最终学会"我可以选择原谅，但不是为了他"。

记忆点标签：#温柔刀 #白切黑 #破碎又锋利
```

**配角小传示例**：
```
【男主】JACK / 杰克（35岁）

外在：现任CEO，Sarah前夫的竞争对手公司。
永远整洁的定制西装，但领带永远松一半——
暗示他"表面上控制一切，内心有脱缰的一面"。

内在：表面冷酷无情，实则有过被背叛的创伤。
他帮助Sarah是因为在她身上看到了曾经的自己。
不擅表达感情，习惯用行动代替语言。

与女主关系：从"有趣的新人"到"想保护的人"
到"发现她的秘密后更想站在她这边"。
核心冲突：当公司利益与Sarah冲突时，他会怎么选？

记忆点标签：#外冷内热 #沉默骑士 #有自己的伤
```

## 第二部分：正文（分集剧本）

### 正文交付格式原则

正文默认使用**中文短剧行业交付稿的普通文本行格式**。Word/PDF 中出现的视觉缩进、居中、段前段后空白只是屏幕排版效果，不能当成剧本语法要求。样本目录中的稳定写法是：

- 场号 + 时间/内外/地点
- `人物：` 或 `出场人物：`
- `△`/`▲` 动作画面
- `角色名（状态/动作/语气）：台词`
- `角色名VO：`、`角色名（OS）：`、`旁白：` 或 `画外音：`

正式交付时，不要把对白/旁白放进表格、引用框、代码框、分栏或单独的“对白框/旁白框”。本文件里少量代码围栏只用于说明文档展示，实际输出剧本时不要带围栏。对白、旁白、动作都按正文同级普通文本行排列。除非用户明确要求“标准美式 screenplay”，不要把角色名和对白做成美式缩进块，也不要把中文角色名另起一行作注释。

### 场景标注格式

每场戏的开头必须标注集-场号、时间、内外、地点、人物。优先使用以下规范：

```
第 X 集
X.Y 地点 日/夜 内/外
人物：角色A、角色B、角色C
```

样本中也常见这些等价写法，可在整理旧稿时保留：

```
1.1 土路边 日 外
人物：林振华（十岁）、陈如烟（五岁）、陈父、陈母

1-1 日，内，酒店
人物：叶知秋，林玹，柳嫣然

1-1男生宿舍 夜 内
出场人物：郭灿

1-1场景：钟爷家
时间：日  内
人物：钟爷  苏沫  壮汉  手下
```

| 元素 | 格式 | 示例 |
|------|------|------|
| **集-场号** | 集数-场次（从1开始） | `1.1`、`1-1`、`3.2`、`3-2` |
| **日/夜** | 日、夜、黄昏、拂晓、深夜等 | `日`、`夜`、`深夜至拂晓` |
| **内/外** | 内=内景 / 外=外景 | `内`、`外` |
| **地点** | 具体场景名称 | 酒店、宴会厅、暴雨街头、醉仙楼 |
| **人物** | `人物：` 或 `出场人物：` | `人物：叶知秋，林玹` |

### 动作、画面、字幕

动作描述前加 `△` 或 `▲`，与对白明确区分。优先全稿统一用 `△`；`▲` 可用于闪回、重点视觉、特写、转场等需要强调的画面。

| 符号/标注 | 用法 | 示例 |
|------|------|------|
| `△` | 常规动作/画面/环境 | `△车外下着雨。` |
| `▲` | 强调画面/闪回/特写/关键转场 | `▲闪回：陆山海睁开眼，眼眶红了。` |
| `【地点/段落】` | 场内小地点或特殊段落 | `【酒店房间】`、`【闪回。日，外，草地。】` |
| `字幕：` / `【字幕：...】` | 屏幕文字 | `【字幕：三个月后】` |

**示例**：
```
1-1 日，内，酒店
人物：叶知秋，林玹，柳嫣然

【酒店门口】
△叶知秋拉着拉杆箱路过酒店门口，高兴地打电话。
叶知秋：老公，我回来了，在哪呢？

△林玹一边亲昵地搂着柳嫣然走进酒店，一边接电话。
林玹（温柔）：今天公司系统升级，要通宵加班，你早点休息。

△叶知秋抬头看见二人，又惊又气，尾随进酒店。
```

### 字幕标注

国内短剧大量使用字幕来快速传递信息。字幕用 `字幕：` 标注：

| 字幕类型 | 示例 |
|---------|------|
| 人物介绍 | `字幕：Sarah 前企业高管 现隐姓埋名实习生` |
| 地点标注 | `字幕：曼哈顿 董事会会议室` |
| 时间提示 | `字幕：六个月后` |
| 地点切换 | `字幕：749局总部` |
| 身份揭示 | `字幕：Jack 现任CEO Sarah前夫的竞争对手` |
| 章节提示 | `字幕：749局月度考核` |
| 倒计时 | `字幕：考核剩余两分钟` |

### 对白、旁白、内心独白标注

正文对白使用**行内冒号格式**：

```
角色名：台词
角色名（语气/动作/状态）：台词
```

旁白与内心独白也保持普通文本行，不使用独立框或缩进块：

| 标注 | 用途 | 示例 |
|------|------|------|
| `角色名（OS）：` | 角色内心独白；观众听见，剧中人不知道 | `姜晚（OS）：歪到刺眼，再看一眼我能当场窒息。` |
| `角色名OS：` | 同上，适合保留样本旧稿写法 | `陆山海OS：哥就是山，压不垮。` |
| `角色名VO：` | 回忆声、画外叙述、角色主观旁白 | `林玹VO：一生一世。` |
| `角色名（VO，语气）：` | 带语气的画外/旁白 | `赵曼琳（VO，冷漠）：陆山海，这婚我不结了。` |
| `旁白：` / `画外音：` | 非角色叙述或环境声 | `画外音：一阵甜腻的声音。` |
| `角色名（用心声传递给X）：` | 读心/传音/特殊能力 | `慕容珩（用心声传递给璃宝）：切记，装疯这事绝不能让旁人知晓。` |

**规则**：
- 角色名后直接接中文全角冒号 `：`。
- 语气、动作、状态放在角色名后的括号里，不另起一行。
- `OS/VO` 可按样本不加点；如面向英文制作团队也可写 `V.O.`，但仍放在同一行。
- 同一稿中 `VO`、`OS` 用法要稳定，不混用“内心独白”和“真实画外说话”。若担心歧义，直接写 `内心` 或 `画外`。
- 括号里的动作可以放在台词前，也可以按样本放在台词中或台词后，例如 `刘伟：（把林琳提起来）...`、`陈如烟：哥哥...（泪水滑落）`。

### 分集剧本完整格式

```
第 X 集
X.Y 地点 日/夜 内/外
人物：角色A、角色B

【可选小地点/段落】
△动作/画面/环境描述（可含景别、灯光、色温、构图、运镜）。
角色A（语气/动作）：台词。
角色B：台词。
角色A（OS）：内心独白。
角色A VO：回忆/旁白式声音。

【闪回。时间，内/外，地点。】
△闪回动作。
角色C VO：闪回里的声音。

字幕：可选屏幕文字/时间提示/身份提示

【本集反转】：[一句话描述本集核心反转]
【下集悬念】：[一句话钩子，引导点击下一集]
【情绪标签】：[虐/甜/燃/泪/惊/气愤，选1-2个]
```

### 完整示例：第1集

```
第1集
1.1 曼哈顿顶层公寓 夜 内
人物：Sarah、Jack

字幕：THE CONTRACT WIFE
△冷钢蓝色调，落地玻璃窗将曼哈顿夜景倾泻进房间。MCU中近景，伦勃朗光从窗外切进来，Sarah半张脸隐在阴影里。
Sarah（OS）：I didn't come here to fall in love. I came here to survive.

△Sarah站在落地窗前，手中握着一杯威士忌。她没有转身，像在审视一座即将被攻克的城市。
字幕：Sarah 前企业高管 现隐姓埋名实习生

△门开了。Jack走进来，定制西装，领带松开，没道歉迟到。
Jack：Sign it. Or don't. But the offer expires at midnight.

△Sarah走向餐桌，那里放着一份合同。她的步伐不急不缓。
Sarah（冷静）：One year. No lies. No touching my company.
Jack（笑）：Still negotiating after midnight?
Sarah：I'm just getting started.

▲ECU大特写：笔尖触及纸面，窗外一道闪电划过她的眼眸。这不是签字，这是宣战。
旁白：I had sixty seconds to decide. I took three.

【本集反转】：Sarah签下合同，但她真正想要的不是婚姻，而是借Jack的局反杀前夫。
【下集悬念】：合同最后一页出现附加条款——"双方不得隐瞒重大身份"。
【情绪标签】：惊、燃
```

## 格式检查清单

### 前述检查
- [ ] 剧名（英文 + 中文）
- [ ] 简介（200-500字，讲清楚人物+冲突+走向）
- [ ] 看点（3-5条，每条一句话，突出卖点）
- [ ] 人物小传（每个角色五维度：外在/内在/状态/行动/变化）
- [ ] 人物关系清晰（谁和谁什么关系）
- [ ] 每个角色有记忆点标签

### 正文检查
- [ ] 每集标注"第 X 集"
- [ ] 每场戏标注"X-Y 内/外 日/夜 地点"
- [ ] 每场戏标注"出场人物"
- [ ] 动作描述前加 `▲` 或 `△` 符号
- [ ] 字幕用 `字幕：` 标注
- [ ] OS（内心独白）和 V.O.（旁白）严格区分
